Creep.js

A tiny jQuery plugin for smooth scrolling anchor links.

Smooth Scrolling Simplified

This plugin makes functioning anchor links smooth scroll to their destinations. With a little configuration, it's simple to target the affected links more specifically to avoid conflicts with other plugins.

All you need is jQuery 1.10+ (or legacy 1.4.0+ - get latest) and a copy of jquery.creep.min.js to get started.

Check the wiki for more documentation of how to use Creep.js
